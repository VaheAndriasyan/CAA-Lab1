# Լաբորատոր աշխատանք - RLE սեղմում

Այս ծրագիրը իրականացնում է տվյալների սեղմում Run-length encoding (RLE)՝ հերթականնությունների երկարությունների կոդավորում [cite: 7] ալգորիթմով։ Գրված է Python-ով։

## Ինչպես օգտագործել

1. Ստեղծել `input.txt` ֆայլ ճիշտ **նույն պապկայի մեջ**, որտեղ գտնվում է Python կոդը։
2. Մեջը գրել տեքստ, որտեղ կան կրկնվող տառեր, օրինակ՝ `AAAAAAAABCCCC`:
3. Աշխատացնել սկրիպտը։
4. Տերմինալում ընտրել `1` (սեղմելու համար) կամ `2` (ապակոդավորելու/հետ բերելու համար)։
5. Ծրագիրը կվերագրանցի ֆայլը նոր արդյունքով։
