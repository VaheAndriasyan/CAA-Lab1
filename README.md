# Լիցենզավորման համակարգ

## Նկարագրություն
Այս նախագիծը իրականացնում է լիցենզավորման համակարգ C++ լեզվով, որը հիմնված է ապարատային ապահովման եզակի բնութագրիչների վրա (CPU ID, MAC հասցե, մայր պլատայի համար):

## Հնարավորություններ
- Մշտական լիցենզիաների ստեղծում
- Ժամանակավոր լիցենզիաների ստեղծում (30 օր)
- Լիցենզիաների ստուգում
- Լիցենզիաների պահպանում ֆայլում

## Պահանջներ
- C++11 կամ ավելի բարձր
- Windows OS (WMI-ի և Windows API-ի կիրառման համար)
- CMake 3.10 կամ ավելի բարձր
