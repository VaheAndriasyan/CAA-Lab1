## 5. Կարճ բացատրություն

**System Fingerprint**-ը (համակարգի թվային նույնացուցիչ) սարքավորումների բնութագրերի (CPU, MAC հասցե, կոշտ սկավառակի ծավալ) եզակի համադրությունն է։ 
Այն հիմնականում կիրառվում է ինֆորմացիոն անվտանգության մեջ՝ ծրագրային ապահովման լիցենզավորման (պաշտպանություն պատճենումից), հակախաբեության համակարգերում (anti-cheat) և սարքերի աուտենտիֆիկացիայի համար։ 
Հեշավորումը (օրինակ՝ SHA256) թույլ է տալիս պահպանել այս տվյալները ստանդարտացված և գաղտնի տեսքով։ Սակայն պետք է հաշվի առնել, որ որոշ պարամետրեր (օրինակ՝ MAC հասցեն) խոցելի են և կարող են հեշտությամբ փոփոխվել (spoofing)՝ շրջանցելով պաշտպանությունը։
