# Լաբորատոր աշխատանք - RLE սեղմում

Այս ծրագիրը իրականացնում է տվյալների սեղմում Run-length encoding (RLE)՝ հերթականնությունների երկարությունների կոդավորում [cite: 7] ալգորիթմով։ Գրված է Python-ով։

## Ինչպես օգտագործել

1. Ստեղծեք `input.txt` ֆայլ ճիշտ **նույն պապկայի մեջ**, որտեղ գտնվում է Python կոդը։
2. Մեջը գրեք տեքստ, որտեղ կան կրկնվող տառեր, օրինակ՝ `AAAAAAAABCCCC`:
3. Աշխատացրեք սկրիպտը։
4. Տերմինալում ընտրեք `1` (սեղմելու համար) կամ `2` (ապակոդավորելու/հետ բերելու համար)։
5. Ծրագիրը կվերագրանցի ֆայլը նոր արդյունքով։

## Կարևոր նշում

Այս բազային տարբերակը լավ աշխատում է միայն այն ժամանակ, երբ կան շատ կրկնվող սիմվոլներ։ Եթե փորձենք սեղմել սովորական տեքստ (որտեղ տառերը չեն կրկնվում), ֆայլի չափը կմեծանա, քանի որ ամեն տառի դիմաց ավելանում է նրա քանակը (1 հատ պրոբելը դառնում է `1 `)։ 
Որպեսզի ալգորիթմը ճիշտ աշխատի ցանկացած տեքստի հետ, պետք է ավելացնել հատուկ բայթ, որի առաջին բիտը ցույց է տալիս կրկնվում է , թե ոչ , և առանձնացնել չկրկնվող հաջորդականությունը կրկնվողից։ Այս սկրիպտը գրված է ալգորիթմի բազային տրամաբանությունը ցույց տալու համար։
